Setting up automatic fitting using ROD
======================================


Below are python functions and a jupyter notebook to setup automatic fitting routines for ROD analysis software, which uses an unofficial hack by adding in macros to the init.mac file, allowing specifying calculations to perform when ROD is next started. 

Download and use the autofitting functions python file RODautofitting.py ( :download:`download python function script <RODautofitting.py>` ) jupyter notebook ROD_auto_fitting.ipynb ( :download:`download notebook <ROD_auto_fitting.ipynb>` )
