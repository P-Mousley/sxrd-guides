Reading and plotting HDF5 amd nexus files
=================================================

A selection of jupyter notebooks with python code to help loading, reading and plotting hdf5 and nexus files


notebooks
----------

- `hdf5_volume_slicer`_ - a notebook to create 2d slices from 3d volumes
- `reading_nexus`_ - a notebook to help loading and reading data from nexus files
- `spherical_polar_plots`_ - a notebook to help visualise spherical polar mapping data
